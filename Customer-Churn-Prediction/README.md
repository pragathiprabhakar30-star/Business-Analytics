Customer Churn Prediction Dashboard

This project analyzes customer churn using machine learning and business analytics techniques.

Tools Used:
- Python (Pandas, Scikit-learn)
- Excel Dashboard

Project Description:
The goal of this project is to predict customer churn and identify key factors affecting customer retention.

Key Insights:
- Month-to-month customers have highest churn
- New customers are more likely to leave
- Higher monthly charges increase churn

Business Impact:
This project helps businesses identify high-risk customers and take preventive actions to reduce churn.
